# Journey Map 重构行为基线

更新日期：2026-06-07

## 用途

本文件是 B1 的重构前行为清单。后续 B2 提取领域与导出、C3 拆分编辑器时，可以改变代码位置和视觉实现，但不能无意改变这里记录的数据与交互行为。

自动基线在 B2 起直接编译并导入正式 `src/tools/journey-map` 模块执行，同时继续读取 `src/App.tsx`、`src/tools/journey-map/JourneyMapPage.tsx` 和 `src/features/assistant/JourneyAssistantPanel.tsx` 检查确认后应用等 UI 约束，不要求为测试修改 UI。

## 自动验证范围

运行：

```powershell
node --test tests/journey-map/journey-map-baseline.test.cjs
```

覆盖：

- 默认标题、场景、目标用户、用户目标；
- 默认 5 个阶段及其名称；
- 默认用户行为、触点、痛点、情绪、机会点 5 行；
- stage、row、cell 的 ID 形状、默认值和模板选择；
- 每行始终按阶段 ID 持有完整 cell；
- AI proposal 对已有 ID 的保留、新 ID 生成、阶段名称别名匹配；
- proposal 缺省字段回退、cell 补齐、情绪分值取整并限制到 1-5；
- 情绪备注中中英文、emoji、换行的计量和 32 单位截断；
- Markdown、JSON、CSV、SVG 的当前数据表示；
- 至少保留一个阶段和一个维度的删除保护；
- 行类型变化时重建该行所有 cell；
- 单元格更新为局部合并，不覆盖同 cell 的其他字段；
- AI proposal 只在用户点击“确认更新”后应用。

代表性夹具位于 `tests/journey-map/fixtures/representative-journey.json`，包含文字、图片、情绪三类行，以及换行、竖线、引号等导出边界。

## 当前输出契约

- 页面入口：`/`
- 组件验收页：`/components`
- 账号页：`/account`，由应用入口路由，不属于 Journey Map 编辑器
- Journey Map 导出：Markdown、JSON、CSV、SVG、打印/PDF
- 下载文件名：旅程标题中的 `\ / : * ? " < > |` 替换为 `-`；空标题回退为 `journey-map`
- Markdown：换行转为 `<br>`，表格竖线转义为 `\|`
- CSV：所有单元格使用双引号包裹，内部双引号写为两个双引号
- 图片行导出图片 URL
- 情绪行导出为 `{score}/5 {note}`
- SVG 使用白色独立导出画布，不跟随页面暗色主题
- SVG cell 文本按每行最多 24 个字符、最多 5 行拆分；长图片 URL 也按文本拆行

## AI 确认流程

1. 请求携带 `serviceName`、当前完整 Journey Map、消息历史和可选截图。
2. `clarify` 只显示问题，不产生可应用 proposal。
3. `proposal` 显示摘要和“确认更新 / 继续调整”操作，不自动写入左侧数据。
4. 用户点击“确认更新”后才运行规范化并替换 Journey Map。
5. 应用后同步服务名称，清空当前选中单元格和情绪编辑态，并追加“修改已应用”消息。
6. “继续调整”只预填输入框，不修改 Journey Map。

## B2/C3 手工回归清单

- [ ] 编辑服务名称时，标题同步为“{服务名称}用户旅程图”；空白名称使用“未命名服务”。
- [ ] 可直接编辑目标用户、用户目标、阶段名称、行标题和文字 cell。
- [ ] 新增阶段时，每一行同步增加对应类型的默认 cell。
- [ ] 删除阶段后，所有行移除该 cell；只剩一个阶段时禁止删除。
- [ ] 可新增文字、图片、情绪行；只剩一个维度时禁止删除。
- [ ] 行类型变化时，该行按新类型重建所有阶段 cell。
- [ ] 删除当前选中的阶段或行时，清除选中态和情绪编辑态。
- [ ] 选中 cell 后，其阶段标题和行标题显示位置提示。
- [ ] 阶段表头和维度列在矩阵滚动时保持冻结。
- [ ] 行高仍可通过 Splitter 拖拽，增删行后面板数量同步。
- [ ] 情绪点指针拖拽映射到 1-5，拖动不会误触发普通点击。
- [ ] 情绪备注可直接编辑，最多 32 单位，宽度与高度随内容变化。
- [ ] 矩阵原生滚动、滚轮、键盘滚动仍可用；自定义滚动条与内容位置同步。
- [ ] Chat 输入 Enter 发送、Shift+Enter 换行；加载中不能重复发送。
- [ ] 非图片附件被忽略，图片以 Data URL 预览并随请求发送。
- [ ] AI 提案未确认前不改变 Journey Map；确认后按规范化结果更新。
- [ ] Markdown、JSON、CSV、SVG 下载内容与自动测试一致；打印隐藏应用操作区。
- [ ] `/`、`/components`、`/account` 可直接访问，浏览器前进后退行为正常。
